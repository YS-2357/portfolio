# Compare-AI 보고서

## 프로젝트 개요

Compare-AI는 단일 프롬프트를 여러 LLM에 동시에 전달하고, 결과를 하나의 화면에서 비교/요약해 선택 시간을 줄이기 위한 서비스입니다.

## 문제 정의

- 모델별 응답 특성이 달라 반복 검토 비용이 큼
- 동일 질문에 대한 결과가 분산되어 기록/비교가 어려움

## 설계 포인트

- 멀티 모델 병렬 호출
- 실패 모델이 있어도 전체 흐름이 깨지지 않는 수집 전략
- 사용자 관점에서 바로 비교 가능한 요약 출력

## 구현 및 운영

- 요청/응답 포맷 표준화
- 스트리밍 기반 사용자 피드백 개선
- 배포 이후 재현 가능한 실행/점검 루틴 정리

## 다음 개선

- 비교 기준 가중치 사용자 커스터마이징
- 프롬프트 템플릿 저장/재실행 기능
- 실험 로그 기반 자동 리포트 생성
